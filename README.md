# Magic Cat App

## 项目介绍

Magic Cat是一个基于Vue3和TypeScripe的猫品种应用程序。

## 技术栈

- Vue3
- TypeScript
- Vue Router
- Pinia
- Vant Ui
- Mockjs
- Axios
- SCSS

## 安装

### 克隆项目

```bash
git clone https://github.com/your-username/magic-cat-app.git
```

### 进入项目目录

```bash
cd magic-cat-app
```

### 安装依赖

```bash
npm install
```

### 启动项目

```bash
npm run dev
```
